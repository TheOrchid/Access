## Installation

1. install package

	```php
    composer require orchid/access
	```

1. edit config/app.php

	service provider :

	```php
	 Orchid\Access\Providers\SettingsServiceProvider::class
	```

1. create user\roles table

	```php
	php artisan vendor:publish
	php artisan migrate
	```
